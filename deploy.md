# Init Astro

## clone github repo

```bash
git clone git@github.com:rei0721/jtkj-home-page.git
```

## install node.js

```bash
# windows
winget install nodejs
```

## install pnpm

```bash
npm install -g pnpm
```

## install modules

```bash
pnpm install
```

# Astro Blog Deploy

```bash
curl -sSL https://raw.githubusercontent.com/rei0721/jtkj-home-page/main/deploy.sh
```
